# 页面结构

这份文档记录当前小程序的主要页面结构和功能边界。

## 探索页：
一、页头：两步式引导（设定起点 / 生成任务）+ 当前步骤状态
![alt text](image.png)

二、纯粹模式

前置限制：必须先通过当前定位、搜索地点或附近地点推荐设定“探索点”。设定后，才能生成主题并开始漫步。

当前地图只展示已选探索点和范围圈，不提供“地图中心确认”入口。

1. 主题卡以“任务票据”形式展示主题、描述、地点摘要和 1 个简单任务。
![alt text](image-1.png)

2. 主题方向选择

随机：作为主题方向中的一个选项，由前端从形状、色彩、声音、数字、气味中抽取一个，再调用单主题生成链路。

选择：用户从形状、色彩、声音、数字、气味中只选择 1 个作为主题。

数字漫步定义：

- 视觉联想：形状像数字，例如井盖像 `0`、树枝交叉像 `4`
- 数量统计：画面里有几个，例如找 `3` 片叶子、`3` 个重复元素
- 符号识别：数字的另一种写法，例如 `IV`、汉字“三”、英文 `Three`
- 密码线索：数字作为行动指令，例如门牌号决定步数、密码 `1990`
![alt text](image-2.png)

3. 任务打卡：点击任务后可录入文字、照片、视频或录音，内容可选。点击任务内的“打卡”按钮后，这一站才算完成。“总结与补充”与普通任务同级。
![alt text](image-3.png)

4. 记录与追踪：支持文字、照片、视频、录音。点击“开启地图”开始追踪，展示栏切换为地图；再次点击后停止追踪，展示栏恢复。
![alt text](image-4.png)

5. 漫步完成后，可在足迹详情页生成打卡卡片并保存。若满足条件，会解锁对应成就。

三、进阶模式

前置限制：同样必须先设定探索点，之后才能生成 AI 主题并开始漫步。

1. 个性化选项：天气（晴朗、多云、雨天、大风）、季节（春夏秋冬）、状态（发呆、元气满满、忧郁、愉悦、未知）、偏好（自然景观、人文历史、市井烟火）。
2. 主题卡以“任务票据”形式展示主题、描述和 3 个任务。
3. AI 生成会结合选项、主题方向、时间和探索点 AOI 语境。
4. 主题方向支持选择 1 到 2 个；选择 2 个时走组合主题生成。
5. 任务打卡。
6. 记录与追踪。
7. 漫步完成后，可生成打卡卡片并保存到足迹；满足条件时解锁成就。


## 足迹页：
分为“纪念卡册”与“成就”两个模块

1. 纪念卡册展示个人单人记录与同行记录，目前为文字卡片列表。
2. 支持按“状态 + 记录类型”组合筛选：状态为全部、待出发、进行中、已完成；类型为全部、单人、同行
3. 筛选区分为“状态”和“记录类型”，并显示筛选摘要与结果条数。
4. 单人记录主要展示进行中和已完成；同行记录还会展示待出发房间。
5. 卡片展示主题、地点、时长/距离、状态、摘要和时间。
6. 点击卡片进入详情。进行中记录可继续任务；已结束单人记录可在详情页分享给微信朋友。
7. 当前用户删除已结束同行记录时，只会从自己的纪念卡册和成就统计中移除，不影响其他成员。
8. 成就模块展示当前用户的成就列表，并可进入成就详情页。
9. 足迹页、探索页、个人页主背景统一为 `#f5f2ed`。

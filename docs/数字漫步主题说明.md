## 2. 数字漫步定义

数字漫步不是单纯“找阿拉伯数字”，而是围绕以下 4 类观察方式展开：

1. 视觉联想
   - 形状像数字。
   - 例如：井盖像 `0`，树枝交叉像 `4`，栏杆或窗格像 `8`。

2. 数量统计
   - 画面里有几个。
   - 例如：找 `3` 片叶子、`3` 只猫、`3` 个重复元素。

3. 符号识别
   - 数字的另一种写法。
   - 例如：`IV`、`VI`、汉字“三”、英文 `Three`。

4. 密码线索
   - 数字作为行动指令。
   - 例如：密码 `1990`、门牌号决定步数、楼层号决定观察顺序。
